# DLR Futures Screener

## Overview
The DLR Futures Screener is a powerful tool designed to help traders identify and analyze futures contracts in the DLR market with efficiency and accuracy.

## Features
- Real-time market data analysis
- Custom alerts and notifications
- User-friendly interface
- Comprehensive reporting

## Installation
To install the DLR Futures Screener, clone the repository and install the required dependencies:

```bash
git clone https://github.com/GaloSzlomowicz/screener-futuros-dlr.git
cd screener-futuros-dlr
npm install
```

## Usage
1. Run the application using the following command:
   ```bash
   npm start
   ```
2. Follow the on-screen instructions to configure your settings.
